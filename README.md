# Ad Mortem - Permadeath

* [Nexus Page](https://www.nexusmods.com/skyrimspecialedition/mods)

### Requirements

* [XMake](https://xmake.io/) [2.8.2+]
* C++23 Compiler (MSVC)

## Build
```
xmake build
```
